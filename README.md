# Vue.jsテストプロジェクト

## 概要
このプロジェクトはVue.jsの勉強用です。
JSON形式のデータを手軽に作成するために作成しました。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
